Project Overview

This experiences is part of my participation in the Models in GIS taught by Dr. Boglárka Bertalan-Balázs course as an exchange student. My final project involved a model using ArcGIS Pro ModelBuilder applied to forestry and land use. This workflow automates a city-by-city forest cover assessment in ArcGIS ModelBuilder. It filters land-cover classes corresponding to forest (e.g., 311, 312, 313), iterates through each administrative boundary, selects only forest polygons fully within each city, dissolves them into a single forest area, and joins city area attributes to compute forest percentage ((forest area / city area) × 100). A companion Python script then consolidates all generated attribute tables from the geodatabase into one Excel report for fast comparison and reporting.
